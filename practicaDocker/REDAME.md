## Descripción
Construye una aplicación sencilla en Spring Boot (que envíe un saludo e indique tu nombre y la calificación que esperarías sacar en esta práctica); realizando lo siguiente:

- Generar el proyecto Spring Boot (añade dependencia Spring Web)
- Crear el controlador
- Git y GitHub
- Dockerizar la aplicación (Dockerfile)
- Subir imagen a Docker Hub
## Entregable:

- Enlace de tu repositorio de GitHub
- Enlace de tu imagen en Docker Hub
- Imagen de ejecución correcta


## Criterios de evaluación:

- Funcionalidad Spring: La aplicación compila y muestra el saludo con el nombre y la nota esperada.
- Gestión de código: El repositorio en GitHub existe, tiene el código fuente completo y al menos un commit descriptivo.
- Dockerización: El Dockerfile está correctamente configurado (usa una imagen base de Java y copia el .jar adecuado).
- Distribución en la nube: La imagen está disponible en Docker Hub y el nombre de la imagen sigue el formato (usuario/nombre-app:tag).
